# Changelog

## 0.8.1

**文件模板（重做）**
- 接入 Obsidian 核心「模板」插件：新建文件节点时用指定模板生成内容
- 新建文件的存放文件夹可配置，默认在当前画布所在文件夹
- 双击画布（双击类型设为「文件」时）直接新建模板笔记并进入编辑状态
- 支持模板变量 `{{title}}` / `{{date}}` / `{{time}}`

**概览模式**
- 标题颜色跟随节点颜色，保留卡片配色
- 默认开启

**性能**
- 多卡片画布打开更流畅

**思维导图**
- 编辑文本时 Enter / Tab 正常换行 / 缩进，不再误创建节点
- 只读画布不再被快捷键创建节点

**设置与本地化**
- 设置页重组为四个标签页：基础设置 / 思维导图 / 节点与边 / 进阶功能
- 设置项全面中文化，术语更易懂（「A* 寻路」→「自动避障」、「Z 轴层级」→「图层顺序」等）
- 命令、右键菜单、导出对话框等界面文本全面中文化
- 补全图片导出、只读模式（缩放 / 平移 / 弹出菜单）的设置项

**其他**
- 移除演示模式（PPT）功能
- 使用指南更名为 guide.md 并全面重写

## 0.7.1

- 新增概览模式：缩小画布时，节点自动显示大号标题填满卡片，放大后恢复原始内容
- 概览模式支持文件节点（显示文件名或第一行标题）和文本节点（显示第一个标题行）
- 概览模式下分组标签自动反向缩放，保持可读性
- 缩放阈值、标题来源、标签缩放均可在设置中自定义
- 新增标题最大字号限制，避免大卡片上的短标题被放大到撑满卡片
- 设置页顶部新增使用指南、更新日志、Star 快捷入口
- 修复多画布同时打开时概览模式只对一个画布生效的问题
- 修复关闭画布后概览模式状态未清理的问题
