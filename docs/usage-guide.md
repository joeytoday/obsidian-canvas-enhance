# Canvas Enhance 使用指南

## 快速开始

1. 在 Obsidian 设置 → 第三方插件中启用 Canvas Enhance
2. 打开或创建一个 `.canvas` 文件
3. 概览模式、节点/边样式、思维导图等默认开启，其余功能在设置中按需开启

## 概览模式

缩小画布时，节点内容变得难以阅读。概览模式自动用大号标题填满每张卡片，让你在全局视角下也能快速识别节点。

### 开启方式

概览模式默认开启。如需关闭：设置 → 基础设置 → 概览模式 → 关闭开关

### 工作原理

- 画布缩放低于阈值（默认 40%）时自动进入概览模式
- 文件节点显示文件名或第一行标题（可在设置中选择）
- 文本节点显示第一个 Markdown 标题（`# 标题`、`## 标题` 等），没有标题则显示第一行文字
- 分组标签自动放大，保持可读
- 放大画布超过阈值后，自动恢复原始内容

### 设置项

| 设置 | 说明 | 默认值 |
|------|------|--------|
| 触发缩放阈值 | 画布缩放低于此值时进入概览模式 | 0.4（40%） |
| 文件节点标题来源 | 文件名 或 第一行标题 | 文件名 |
| 分组标签反向缩放 | 缩小时自动放大分组标签 | 开启 |
| 标题最大字号 | 标题文字的最大字号，避免大卡片标题过大 | 64 |

## 文件模板

开启后，新建文件节点会用指定的 Obsidian 模板生成文件内容。

### 使用方式

1. 先启用 Obsidian 核心「模板」插件，并设置模板文件夹
2. 设置 → 基础设置 → 文件模板 → 打开开关
3. 在「选择模板」下拉中选一个模板
4. 之后新建文件节点时，自动用该模板填充内容

支持模板变量：`{{title}}`（文件名）、`{{date}}`（日期）、`{{time}}`（时间）。模板只对**新建的空文件**生效，拖入已有文件不受影响。

## 思维导图

将画布变成思维导图工具。

- `Tab` 创建下级节点
- `Enter` 创建同级节点
- `Alt + 方向键` 在节点间导航
- 下级节点自动继承上级节点颜色

设置 → 思维导图 标签页中可调整节点间距和快捷键。

## 节点样式

通过右键菜单为节点添加流程图形状：

- 终端（圆角矩形）
- 流程（矩形）
- 判断（菱形）
- 数据库（圆柱）
- 文档、平行四边形、圆形等

还支持边框样式（虚线/点线）和文本对齐。

## 边样式

右键菜单中可设置：

- 路径样式：实线、点线、短虚线、长虚线
- 箭头样式：8 种可选
- 连线走向：曲线、直线、直角折线、自动避障

## 自定义 CSS 样式

通过 CSS snippet 扩展节点和边的样式选项。在 CSS 文件中用 YAML 注释定义样式，即可在弹出菜单中使用。

```css
/* @canvas-enhance-node-style
key: validation-state
label: 验证状态
options:
  - label: 无状态
    value: null
    icon: circle-help
  - label: 已通过
    value: approved
    icon: circle-check
*/
```

详见 [README 自定义样式章节](https://github.com/joeytoday/obsidian-canvas-enhance?tab=readme-ov-file#自定义样式)。

## 其他功能

- **元数据缓存**：`.canvas` 文件接入 Obsidian 元数据系统，支持关系图谱和反向链接
- **单节点链接**：用 `[[canvas#node-id]]` 链接或嵌入 Canvas 中的单个节点
- **画布内搜索**：`Ctrl+F` 搜索 Canvas 中的文本
- **文件模板**：新建文件节点时用指定的 Obsidian 模板填充内容
- **可折叠组**：组节点可折叠/展开
- **聚焦模式**：模糊未选中节点
- **图片导出**：导出为 PNG/SVG
- **图层顺序**：右键菜单调整节点前后叠放顺序
- **传送门**：在 Canvas 中嵌入另一个 Canvas 的视图

## 常见问题

**Q: 功能没有生效？**
A: 部分功能默认关闭，需要在设置中手动开启；开启后若仍无效，尝试重新加载 Obsidian。

**Q: 概览模式没有触发？**
A: 检查缩放阈值设置。默认 0.4 表示缩小到 40% 以下才触发。如果你希望更早触发，可以调高这个值。

**Q: 如何反馈问题？**
A: 在 [GitHub Issues](https://github.com/joeytoday/obsidian-canvas-enhance/issues) 提交 bug 报告或功能建议。
