# Canvas Enhance 使用指南

## 快速开始

1. 在 Obsidian 设置 → 第三方插件中启用 Canvas Enhance
2. 打开或创建一个 `.canvas` 文件
3. 概览模式、节点/边样式、思维导图等默认开启，其余功能在设置中按需开启

设置分为四个标签页：**基础设置**、**思维导图**、**节点与边**、**进阶功能**。

---

## 概览模式

缩小画布时，节点内容变得难以阅读。概览模式自动用大号标题填满每张卡片，让你在全局视角下也能快速识别节点；放大画布超过阈值后自动恢复原始内容。

> 📹 待补充演示：缩小画布触发概览模式、放大恢复的过程

- 文件节点显示文件名或第一行标题（可在设置中选择）
- 文本节点显示第一个 Markdown 标题（`# 标题`），没有标题则显示第一行文字
- 有颜色的节点，标题会跟随节点颜色
- 分组标签自动放大，保持可读

**设置项**（基础设置 → 概览模式）：

| 设置 | 说明 | 默认值 |
|------|------|--------|
| 触发缩放阈值 | 画布缩放低于此值时进入概览模式 | 0.4（40%） |
| 文件节点标题来源 | 文件名 或 第一行标题 | 文件名 |
| 分组标签反向缩放 | 缩小时自动放大分组标签 | 开启 |
| 标题最大字号 | 标题文字的最大字号，避免大卡片标题过大 | 64 |

## 文件模板

新建文件节点时，用指定的 Obsidian 模板生成文件内容，并存放到指定文件夹。

> 📹 待补充演示：双击新建模板笔记、自动填充模板内容的过程

**使用方式**：

1. 先启用 Obsidian 核心「模板」插件，并设置模板文件夹
2. 设置 → 基础设置 → 文件模板 → 打开开关
3. 在「选择模板」下拉中选一个模板
4. 在「新建文件所在文件夹」选择存放位置（留空＝当前画布所在文件夹）

之后**双击画布空白处**（双击类型设为「文件」时）或运行命令「创建模板笔记节点」，即可以该模板新建笔记卡片，并自动进入编辑状态。

支持模板变量：`{{title}}`（文件名）、`{{date}}`（日期）、`{{time}}`（时间）。

## 思维导图

将画布变成思维导图工具，用快捷键快速搭建结构：

> 📹 待补充演示：Tab/Enter 创建节点、方向键导航、颜色继承

- `Tab` 创建下级节点
- `Enter` 创建同级节点
- `Alt + 方向键` 在节点间导航
- `Space` 编辑选中节点
- 下级节点自动继承上级节点颜色

设置 → 思维导图 标签页中可调整节点间距、导航快捷键和颜色传播。

## 节点样式

通过右键菜单为节点添加流程图形状和外观：

![流程图形状](../assets/docs/sample-flowchart.png)

- **形状**：圆角矩形、胶囊、菱形、平行四边形、圆形、预定义流程、文档、数据库
- **边框**：实线、虚线、点线、隐藏
- **文本对齐**：左对齐、居中、右对齐

> 📹 待补充演示：右键菜单切换节点形状

## 边样式

右键点击边，可设置连线的样式：

![边路径样式](../assets/docs/edge-path-styles.png)

- **路径样式**：实线、点线、短虚线、长虚线
- **箭头样式**：8 种可选（三角形、菱形、圆形及其轮廓等）
- **连线走向**：曲线、直线、直角折线、自动避障

![边箭头样式](../assets/docs/edge-arrow-styles.png)

「自动避障」会让连线绕开中间的节点，适合复杂的流程图。

## 自定义 CSS 样式

通过 CSS snippet 扩展节点和边的样式选项。在 CSS 文件中用 YAML 注释定义样式，即可在右键菜单中使用。

![自定义样式属性](../assets/docs/custom-style-attribute-example.png)

```css
/* @canvas-enhance-node-style
key: validation-state
label: 验证状态
options:
  - label: 无状态
    value: null
    icon: circle-help
  - label: 已通过
    value: approved
    icon: circle-check
*/
```

详见[示例 CSS 文件](../assets/docs/example-custom-node-style.css)。

## 节点操作

### 自动调整大小

文本/文件节点根据内容自动调整高度，内容变多时节点自动变高。

![自动调整大小](../assets/docs/auto-node-resizing.gif)

设置 → 节点与边 → 自动调整节点大小，可配置默认启用、最大高度、对齐网格。

### 图层顺序

调整节点的前后叠放顺序（谁压在最上面）。右键节点 → 置于顶层 / 置于底层，开启「显示上移/下移选项」后还可逐层移动。

![图层顺序](../assets/docs/z-ordering-control.png)

### 可折叠分组

组（group）节点可以折叠/展开，折叠时把内部节点收起，让画布更清爽。

![可折叠分组](../assets/docs/collapsible-groups.png)

### 宽高比控制

调整节点大小时锁定宽高比，避免图片节点变形。右键节点 → 设置宽高比。

## 边操作

### 浮动边

边会自动选择连接到节点的哪一侧（上下左右），拖动节点时连接点智能跟随；还支持不连接到节点的「浮动边」。

![浮动边](../assets/docs/floating-edge-example.png)

### 翻转边

一键翻转边的方向（A→B 变成 B→A）。右键边 → 翻转边。

![翻转边](../assets/docs/flip-edge.gif)

### 边高亮

选中节点时高亮与它相连的边，看清连接关系。可选择只高亮出边或同时高亮入边。

![边高亮](../assets/docs/select-connected-edges.gif)

## 画布功能

### 元数据缓存

让 `.canvas` 文件接入 Obsidian 的元数据系统：关系图谱里能看到画布、画布里的 `[[链接]]` 会出现在对应笔记的反向链接和出链中。

![元数据缓存](../assets/docs/metadata-cache-support.png)

### 单节点链接

用 `[[画布名#节点ID]]` 链接或嵌入画布里的**单个节点**（而不是整张画布）。右键节点 → 复制节点 Wikilink 获取链接。

### 画布内搜索

在画布里按 `Ctrl/Cmd + F` 搜索文本，逐个高亮并跳转到匹配的节点。

![画布内搜索](../assets/docs/sample-search.gif)

### 图片导出

把整张画布或选中的节点导出为 PNG / SVG 图片，支持像素比、透明背景、主题、隐私模式等选项。命令面板运行「导出画布为图片」。

> 📹 待补充演示：导出画布为图片

### 传送门

在一个画布里嵌入**另一个画布**的视图，像传送门一样在画布间嵌套浏览。

![传送门](../assets/docs/sample-portal-usage.png)

### 专注模式

模糊/暗化未选中的节点，突出当前选中的节点，适合聚焦局部内容。命令面板运行「切换专注模式」。

![专注模式](../assets/docs/focus-mode.png)

### 封装选区

把选中的多个节点一键封装成一个组。选中节点后运行「封装选区」命令。

> 📹 待补充演示：封装选区成组

### 增强只读

增强 Obsidian 的只读模式：只读下可隐藏背景网格、隐藏节点弹出菜单、锁定缩放和平移，防止误操作。

## 常见问题

**Q: 功能没有生效？**
A: 部分功能默认关闭，需要在设置中手动开启；开启后若仍无效，尝试重新加载 Obsidian。

**Q: 概览模式没有触发？**
A: 检查缩放阈值设置。默认 0.4 表示缩小到 40% 以下才触发。如果你希望更早触发，可以调高这个值。

**Q: 文件模板下拉是空的？**
A: 需要先启用 Obsidian 核心「模板」插件并设置模板文件夹，下拉才会列出模板。

**Q: 如何反馈问题？**
A: 在 [GitHub Issues](https://github.com/joeytoday/obsidian-canvas-enhance/issues) 提交 bug 报告或功能建议。
